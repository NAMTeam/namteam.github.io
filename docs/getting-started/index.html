Compatibility Notice - EA App and Origin Copies of SimCity 4 Deluxe NOW COMPATIBLE with the Network Addon Mod

Digital copies of SimCity 4 Deluxe purchased via EA.com, the EA App, or the Origin Store have been patched to Version 1.1.641 as of 1 June 2023, and are now fully compatible with the Network Addon Mod. The same is true if playing the game via an EA Play or Xbox Game Pass subscription.
