Compatibility Warning - EA App and Origin Copies of SimCity 4 Deluxe

Digital copies of SimCity 4 Deluxe purchased via EA.com, the EA App, or the Origin Store feature an unpatched executable that is below the minimum requirements for this mod. The same is true if playing the game via an EA Play or Xbox Game Pass subscription.
